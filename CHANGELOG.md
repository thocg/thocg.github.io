# Changelog

Format follows Keep a Changelog; dates use ISO 8601 (YYYY-MM-DD).

- New "O" section: add "Operational Technology Security (OT Security)".
- Add requested terms: "API Security", "Supply Chain Attack (Third-Party Attack)", "Ransomware-as-a-Service (RaaS)", "DevSecOps", "Shadow IT".
- Add related terms: "Business Email Compromise (BEC / CEO Fraud)", "Credential Stuffing", "Endpoint Detection and Response (EDR)", "Extended Detection and Response (XDR)", "Lateral Movement", "Multi-Factor Authentication (MFA)", "Principle of Least Privilege (PoLP / Least Privilege)", "Privilege Escalation (Elevation of Privilege, EoP)", "Vishing (Voice Phishing)", "Zero Trust Network Access (ZTNA)".
- Total terms: 198.
